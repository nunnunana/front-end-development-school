### outline 알고리즘

**초기 단계**
```html
<section>
  <h1>Main Headline</h1>
  <!-- <section> 암묵적으로 생기는 section -->
    <h2>Sub Headline</h2>
  <!-- </section> 끝 -->
</section>
```

**hgroup**
```html
<section>
  <hgroup>
    <h1>Main Headline</h1>
    <section>
      <h2>Sub Headline</h2>
    </section>
  </hgroup>
</section>
```
